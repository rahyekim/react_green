자바스크립트의 맵은 다른프로그래밍 언어 파이썬 딕셔너리와
    유사하게 key-value 쌍의 모음을 저장할 수있는 객체
    만드는 법 new Map([])

    메서드
    .get(): 특정키에 값을 가져옴
    .set(): 요소를 추가
    .size(): 요소의 개수를 리턴
    .delete(): 요소를 삭제
    .clear(): 바깥 껍데기는 남겨놓고 안에 내용만 없애줌 외형만존재
    .has(): 키가 존재하면 true를 반환
    .forEach(): 맵의 각 키/ 값에 대해 쌍으로 콜백함수 호출
    .entries(): 키 값이 저장된 반복자 객체를 리턴
    .keys(): 키값만 반환
    .values(): 값만 나오게
